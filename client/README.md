### Dev Setup

#### install dependencies
```
npm install

# in case you encounter errors installing dependencies

npm install --legacy-peer-deps
```

#### create a .env file and add
```
URL=[your local ip address]
```
#### serve with hot reload at localhost:4100
```
- ./start.sh
```
